## 二分查找基础

二分查找是一种在**有序序列**上以 $O(\log n)$ 时间查找目标的算法。

### 整数二分模板

```cpp
// 查找满足条件的最小值
int binary_search(int lo, int hi) {
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (check(mid)) {
            hi = mid;
        } else {
            lo = mid + 1;
        }
    }
    return lo;
}
```

### 浮点数二分模板

```cpp
double binary_search(double lo, double hi) {
    for (int i = 0; i < 100; i++) {  // 迭代 100 次足够精确
        double mid = (lo + hi) / 2;
        if (check(mid)) {
            hi = mid;
        } else {
            lo = mid;
        }
    }
    return lo;
}
```

## 常见应用场景

1. **精确查找**：在有序数组中找目标值
2. **边界查找**：找第一个大于等于 target 的位置
3. **答案二分**：最大化最小值 / 最小化最大值
4. **浮点二分**：求方程的近似解

## 例题

### 例题 1：数的范围

> 给定一个升序数组，查询某个数的起始位置和结束位置。

思路：两次二分，分别找左边界和右边界。

### 例题 2：分割数组的最大值

> 将数组分成 m 段，使各段和的最大值最小。

思路：二分答案，`check(mid)` 判断能否分成不超过 m 段。

## 注意事项

- 二分的边界条件容易出错，建议固定使用一套模板
- `mid = lo + (hi - lo) / 2` 可以防止溢出
- 答案二分时，`check` 函数的设计是关键
